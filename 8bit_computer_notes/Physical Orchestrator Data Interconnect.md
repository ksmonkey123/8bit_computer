[[Physical Bus]] providing the orchestrator with access to the [[Data Bus]] and the [[Status Flag]] status.

|   Pin | Function           |
| ----: | ------------------ |
|     1 | data bus bit 0     |
|     2 | data bus bit 1     |
|     3 | data bus bit 2     |
|     4 | data bus bit 3     |
|     5 | data bus bit 4     |
|     6 | data bus bit 5     |
|     7 | data bus bit 6     |
|     8 | data bus bit 7     |
|     9 | $F_C$              |
|    10 | $F_Z$              |
|    11 | $F_S$              |
|    12 | $\overline{reset}$ |
| 13,14 | VCC (+5V)          |
| 15,16 | GND                |
