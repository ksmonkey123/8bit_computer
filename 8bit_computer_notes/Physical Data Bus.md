[[Physical Bus]] primarily tasked with holding the [[Data Bus]], power and some control signals.


|   Pin | Function                              |
| ----: | ------------------------------------- |
|     1 | data bus bit 0                        |
|     2 | data bus bit 1                        |
|     3 | data bus bit 2                        |
|     4 | data bus bit 3                        |
|     5 | data bus bit 4                        |
|     6 | data bus bit 5                        |
|     7 | data bus bit 6                        |
|     8 | data bus bit 7                        |
|     9 | $\overline{readMemory}$               |
|    10 | $\overline{writeMemory}$              |
|    11 | $\overline{reset}$ ([[System Reset]]) |
|    12 | $clock$                               |
| 13,14 | VCC (+5V)                             |
| 15,16 | GND                                   |
